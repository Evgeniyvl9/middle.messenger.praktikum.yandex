## Установка

### Установка стабильной версии
```sh
npm install
```

### Сборка стабильной версии и запуск сервера

```sh
npm start
```

### Компиляция и горячая перезагрузка для разработки

```sh
npm run dev
```

### Сборка стабильной версии

```sh
npm run build
```
